About Moony Creator Project

VISION

Our vision is to create Moony Creator which is:

This project aims to create a file system utility which can be used for creating and manipulating different properties of file system.

Currently limited support of filesystems are present, but we focus on increasing it's number.

If you you are interested in joining the development, or contributing in any other way, feel free to contact us at psibi2000@gmail.com

HISTORY

Moony Created started as a hobby project during the holidays. After initially being used up by some Linux enthusiasts at our university, it became open source project on SourceForge.
