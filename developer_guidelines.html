Developer Gudelines

Here are some guidelines for developers which will help us to manage the project, and keep the quality of code and documentation:

  1. Please put comments for classes and methods in your code so javadoc can generate documentation.

  2. Prefered tool for any GUI related development is Qt Designer

  3. Please follow the Python coding conventions.

  4. Althoug git is not enabled for revision control system, it may be enabled on request.
